# Inscrição em Workshop Fictício

## Descrição

Este projeto simula uma página de inscrição para um workshop fictício. O objetivo é permitir que usuários realizem sua inscrição por meio de um formulário online, com armazenamento automático dos dados e envio de confirmação por e-mail.

## Como Funciona

1. O usuário acessa o site e recebe informações sobre o workshop.

2. O usuário preenche o formulário de inscrição com seus dados.

3. Os dados enviados são armazenados automaticamente em uma planilha do Google Sheets.

4. O Google Apps Script processa a inscrição e envia um e-mail de confirmação para o usuário.

## Tecnologias Utilizadas

* builder.io
* Google Apps Script
* Google Sheets

## Funcionalidades

* Formulário de inscrição online.
* Armazenamento automático dos dados em uma planilha.
* Envio automático de e-mail de confirmação.
* Integração entre formulário, Google Sheets e Google Apps Script.
